# AI Cover Letter Generator

A full-stack application built with React and Express that generates cover letters using the Google Gemini API.

## Features

- Candidate details form
- Optional job description
- Gemini API integration
- Template fallback
- Loading and error states
- Copy to clipboard
- Responsive layout

## Tech Stack

- React
- Vite
- Node.js
- Express.js
- Google Gemini API
